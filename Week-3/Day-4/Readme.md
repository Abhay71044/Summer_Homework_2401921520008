# Week 3 - Day 4 

Today I solved 3 Stack-based problems from LeetCode.

## Problems Solved

### 1. Daily Temperatures
- Used a monotonic stack to keep track of unresolved temperatures.
- Calculated the number of days required to get a warmer temperature.

Time Complexity: O(n)
Space Complexity: O(n)

### 2. Evaluate Reverse Polish Notation
- Used a stack to evaluate postfix expressions.
- Performed arithmetic operations whenever an operator was encountered.

Time Complexity: O(n)
Space Complexity: O(n)

### 3. Largest Rectangle in Histogram
- Used Previous Smaller Element and Next Smaller Element concepts.
- Calculated the maximum rectangular area for each bar.

Time Complexity: O(n)
Space Complexity: O(n)

## Concepts Learned
- Stack
- Monotonic Stack
- Previous Smaller Element
- Next Smaller Element
- Histogram Problems
- Expression Evaluation
