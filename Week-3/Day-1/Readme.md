# Week 3 - Day 1

Today I solved 3 Linked List problems from LeetCode.

## Problems Solved

### 1. Linked List Cycle
- Used Floyd's Cycle Detection Algorithm (Slow and Fast Pointers).
- If slow and fast pointers meet, a cycle exists.

Time Complexity: O(n)
Space Complexity: O(1)

### 2. Reverse Linked List
- Used Recursion to reverse the linked list.
- Changed the direction of links while returning from recursive calls.

Time Complexity: O(n)
Space Complexity: O(n) (Recursion Stack)

### 3. Middle of the Linked List
- Used Slow and Fast Pointer technique.
- Slow pointer reaches the middle when fast pointer reaches the end.

Time Complexity: O(n)
Space Complexity: O(1)

## Concepts Learned
- Linked Lists
- Slow and Fast Pointers
- Floyd's Cycle Detection
- Recursion
- Pointer Manipulation
