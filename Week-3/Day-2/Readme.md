# Week 3 - Day 2

Today I solved 3 Linked List problems from LeetCode.

## Problems Solved

### 1. Merge Two Sorted Lists
- Merged two sorted linked lists into a single sorted linked list.
- Used a dummy node to simplify the merging process.

Time Complexity: O(n + m)
Space Complexity: O(1)

### 2. Remove Nth Node From End of List
- Calculated the length of the linked list.
- Removed the required node using its position from the beginning.

Time Complexity: O(n) 
Space Complexity: O(1)

### 3. Palindrome Linked List
- Found the middle of the linked list using slow and fast pointers.
- Reversed the second half of the list and compared both halves.

Time Complexity: O(n)
Space Complexity: O(1)

## Concepts Learned
- Linked Lists
- Dummy Node Technique
- Slow and Fast Pointers
- List Reversal
- Palindrome Checking
