# Week 4 - Day 4

Today I solved 3 Binary Tree problems from LeetCode.

## Problems Solved

### 1. Path Sum
- Used Recursion to check whether there exists a root-to-leaf path with the given target sum.
- Subtracted the current node value from the target sum while traversing the tree.

Time Complexity: O(n)
Space Complexity: O(h)

### 2. Diameter of Binary Tree
- Used a recursive height function to calculate the diameter.
- Updated the maximum diameter while computing subtree heights.

Time Complexity: O(n)
Space Complexity: O(h)

### 3. Binary Tree Maximum Path Sum
- Used Depth First Search (DFS) to compute the maximum path sum.
- Ignored negative path sums and maintained the global maximum answer.

Time Complexity: O(n)
Space Complexity: O(h)
