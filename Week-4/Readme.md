# Week 4 - Day 1

Today I solved 3 Binary Tree problems from LeetCode.

## Problems Solved

### 1. Maximum Depth of Binary Tree
- Used Recursion to find the depth of left and right subtrees.
- Returned the maximum depth among both subtrees.

Time Complexity: O(n)
Space Complexity: O(h)

### 2. Invert Binary Tree
- Used Recursion to swap the left and right child nodes.
- Inverted the entire tree recursively.

Time Complexity: O(n)
Space Complexity: O(h)

### 3. Same Tree
- Used Recursion to compare corresponding nodes of both trees.
- Checked node values and recursively verified left and right subtrees.

Time Complexity: O(n)
Space Complexity: O(h)
