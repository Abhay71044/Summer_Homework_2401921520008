# Week 4 - Day 2

Today I solved 3 Binary Tree Traversal problems from LeetCode.

## Problems Solved

### 1. Binary Tree Inorder Traversal
- Used Recursion to traverse the tree in Left-Root-Right order.
- Stored the traversal in a vector.

Time Complexity: O(n)
Space Complexity: O(h)

### 2. Binary Tree Level Order Traversal
- Used Queue to perform Breadth First Search (BFS).
- Processed nodes level by level and stored each level separately.

Time Complexity: O(n)
Space Complexity: O(n)

### 3. Binary Tree Zigzag Level Order Traversal
- Used BFS with a direction flag.
- Alternated the insertion order at each level to achieve zigzag traversal.

Time Complexity: O(n)
Space Complexity: O(n)
