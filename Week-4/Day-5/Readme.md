# Week 4 - Day 5

Today I solved 3 Binary Tree problems from LeetCode.

## Problems Solved

### 1. Symmetric Tree
- Used recursion to check whether the left and right subtrees are mirror images of each other.
- Compared corresponding nodes recursively.

Time Complexity: O(n)
Space Complexity: O(h)

### 2. Construct Binary Tree from Preorder and Inorder Traversal
- Used recursion to build the tree from preorder and inorder traversals.
- Identified the root from preorder traversal and divided the inorder traversal into left and right subtrees.

Time Complexity: O(n²)
Space Complexity: O(h)

### 3. Serialize and Deserialize Binary Tree
- Used preorder traversal to serialize the tree into a string.
- Used recursion to reconstruct the tree during deserialization.
- Represented null nodes using a special marker.

Time Complexity: O(n)
Space Complexity: O(n)
