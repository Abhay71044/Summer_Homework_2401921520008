# Week 4 - Day 3

Today I solved 3 Binary Search Tree (BST) problems from LeetCode.

## Problems Solved

### 1. Search in a Binary Search Tree
- Used recursion to search for a given value in the BST.
- Traversed either the left or right subtree based on BST properties.

Time Complexity: O(h)
Space Complexity: O(h)

### 2. Validate Binary Search Tree
- Used recursion with lower and upper bounds.
- Verified that each node satisfies BST conditions.

Time Complexity: O(n)
Space Complexity: O(h)

### 3. Lowest Common Ancestor of Deepest Leaves
- Used Depth First Search (DFS).
- Calculated the depth of left and right subtrees and identified the lowest common ancestor of the deepest leaves.

Time Complexity: O(n)
Space Complexity: O(h)
