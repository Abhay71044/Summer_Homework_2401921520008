# Week 4 - OOPs Assignment

## 📌 Topics Covered
This assignment focuses on Object-Oriented Programming (OOP) concepts in Java.

### Programs Included

1. **Interface Implementation**
   - Created an interface `Test` with the method `square(int n)`.
   - Implemented the interface in the `Arithmetic` class.
   - Used the implemented method in the `ToTestInt` class.

2. **Inner Class**
   - Created an outer class with a `display()` method.
   - Created an inner class having its own `display()` method.
   - Demonstrated calling both methods.

3. **Point Class**
   - Implemented constructors.
   - Used private instance variables `x` and `y`.
   - Added methods:
     - `setX()`
     - `setY()`
     - `setXY()`
   - Demonstrated object creation and updating coordinates.

4. **Box and Box3D**
   - Implemented inheritance.
   - Created `Box` as the base class.
   - Created `Box3D` as the derived class.
   - Calculated:
     - Area of the box
     - Volume of the 3D box
