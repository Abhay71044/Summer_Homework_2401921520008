# Week 1 - Day 4

Today I solved 3 matrix-based problems from LeetCode.

## Problems Solved

### 1. Matrix Diagonal Sum
- Traversed the matrix and added elements belonging to the primary and secondary diagonals.
- Avoided missing the center element in odd-sized matrices.

Time Complexity: O(n²)
Space Complexity: O(1)

### 2. Reshape the Matrix
- Checked whether reshaping was possible.
- Mapped elements from the original matrix to the new matrix row by row.

Time Complexity: O(m × n)
Space Complexity: O(r × c)

### 3. Spiral Matrix
- Used four boundaries (top, bottom, left, right).
- Traversed the matrix in spiral order until all elements were visited.

Time Complexity: O(m × n)
Space Complexity: O(1) (excluding output array)

## Concepts Learned
- Matrix Traversal
- Diagonal Traversal
- Matrix Reshaping
- Boundary-Based Traversal
- Spiral Order Traversal
