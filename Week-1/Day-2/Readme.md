# Week 1 - Day 2

Today I solved 3 problems based on Arrays, Hashing and Sliding Window.

## Problems Solved

### 1. Maximum Subarray
- Used Kadane's Algorithm.
- Found the maximum sum of a contiguous subarray.

Time Complexity: O(n)
Space Complexity: O(1)

### 2. Contains Duplicate
- Sorted the array and checked adjacent elements.
- If two consecutive elements were equal, returned true.

Time Complexity: O(n log n)
Space Complexity: O(1)

### 3. Maximum Average Subarray I
- Used Sliding Window technique.
- Maintained the sum of a fixed-size window and updated the maximum sum.

Time Complexity: O(n)
Space Complexity: O(1)

## Concepts Learned
- Kadane's Algorithm
- Sorting
- Sliding Window
- Time and Space Complexity Analysis
