# Library Management System

## Objective
To understand the concept of Interface in Java and implement it using different types of library users.

## Interface Used
- LibraryUser

## Classes Created
- KidUser
- AdultUser
- LibraryInterfaceDemo

## Functionality

### Kid User
- Can register only if age is less than 12.
- Can borrow only Kids category books.
- Book return period: 10 days.

### Adult User
- Can register only if age is greater than 12.
- Can borrow only Fiction category books.
- Book return period: 7 days.

## Concepts Used
- Interface
- Class and Object
- Method Implementation
- Conditional Statements
- Java OOPs

## Output
The program checks:
- User registration based on age.
- Book issue based on book category.
- Displays appropriate messages for valid and invalid cases.
