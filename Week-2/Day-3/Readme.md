# Week 2 - Day 3

Today I solved 3 pattern matching and string problems from LeetCode.

## Problems Solved

### 1. Find the Index of the First Occurrence in a String (strStr)
- Used a brute force approach to compare the pattern with every possible position in the string.
- Returned the starting index of the first match.

Time Complexity: O(n × m)
Space Complexity: O(1)

### 2. Is Subsequence
- Used Two Pointer technique.
- Traversed both strings and checked if all characters of the first string appeared in order.

Time Complexity: O(n + m)
Space Complexity: O(1)

### 3. Repeated Substring Pattern
- Checked all possible substring lengths.
- Repeated the substring and compared it with the original string.

Time Complexity: O(n²)
Space Complexity: O(n)

## Concepts Learned
- Pattern Matching
- String Traversal
- Two Pointers
- Substring Operations
- Brute Force Approach
