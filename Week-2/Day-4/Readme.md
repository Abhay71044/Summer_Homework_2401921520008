# Week 2 - Day 4

Today I solved 3 recursion and string-based problems from LeetCode.

## Problems Solved

### 1. Reverse Words in a String III
- Traversed the string and reversed each word individually.
- Used the built-in reverse function.

Time Complexity: O(n)
Space Complexity: O(1)

### 2. Decode String
- Used Stack to process encoded patterns.
- Decoded strings by repeating substrings according to the given number.

Time Complexity: O(n)
Space Complexity: O(n)

### 3. Generate Parentheses
- Used Recursion and Backtracking.
- Generated all valid combinations of parentheses.

Time Complexity: O(4ⁿ / √n)
Space Complexity: O(n)

## Concepts Learned
- Recursion
- Backtracking
- Stack
- String Manipulation
- Pattern Decoding
