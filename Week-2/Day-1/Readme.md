# Week 2 - Day 1

Today I solved 3 string hashing problems from LeetCode.

## Problems Solved

### 1. Valid Anagram
- Used a frequency array to count characters.
- Compared character frequencies of both strings.

Time Complexity: O(n)
Space Complexity: O(1)

### 2. First Unique Character in a String
- Used Hash Map to store character frequencies.
- Traversed the string again to find the first unique character.

Time Complexity: O(n)
Space Complexity: O(n)

### 3. Ransom Note
- Used a frequency array to count available characters.
- Checked whether all characters of the ransom note could be formed.

Time Complexity: O(n)
Space Complexity: O(1)

## Concepts Learned
- Hashing
- Frequency Array
- Hash Map
- String Manipulation
- Character Counting
